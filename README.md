# PulseQ Website (Frontend)

Skip the Line — frontend only.

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
npx serve .
```

Then open http://localhost:3000

## Pages

- **Home** — Hero, problem vs solution, features, trust bar, live queue demo
- **Our Solution** — OPD vs clinics, AI, data privacy
- **For Hospitals** — ROI, efficiency, staff management
- **Partners** — Current partners, success stories, become a partner form
- **Blog** — Categories: Patient Experience, Hospital Efficiency, Tech in Health, Company News
- **Contact** — Book a Demo form, support, team section
- **Privacy Policy** & **Terms of Service** — Legal
